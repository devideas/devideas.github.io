Dev Ideas

An email newsletter for dev project ideas.

Sign up for this newsletter if you want a bit of inspiration on what you should build next. You can expect around 3-5 ideas in your inbox every Saturday.

If you have a cool idea you'd like to share, just send a tweet to @rjun07a or send an email to sarodearjun57@gmail.com and I'll add it to the next newsletter.

Some old ideas

  • Anime Spotlight Site

    Create a beautiful web design to show off your favorite anime or movie. The entire page would show information about the anime, your review, etc. It could turn into your personal site for anime/movie reviews.

    Similar: Movie Card

  • The Color of the Date

    Let the user input their birthdate or the date of some event and change the background color to be a unique color based on the date. You could even grab dated information from an API and allow the user to see the color of that item. Ex: Paste a link to a reddit post and display the color of the date the post was made.

    Similar: What Colour Is It

  • ERMAHGERD Subtitle Translator

    Write a script (this might be a good excuse to try out Python) to convert the subtitle of a show to ERMAHGERD. You could also convert it to yoda speak.

    Helpful: ERMAHGERD API

  • Redesign "The Math Club" Site

    I've heard this is a popular math tutor club. You could try to redesign the site so it has a better user experience. while you're at it, you could make it look prettier too.

    Current site: "The Math Club" Site
